# Revenge Relationship Notifier

Root-only upload package for GitHub Pages.

Upload `manifest.json`, `index.js`, and `README.md` directly into the repository root.
Then enable GitHub Pages using **Deploy from a branch**, branch `main`, folder `/ (root)`.

Plugin URL:

https://lapialregalia.github.io/Revenge-Relationship-Notifier/

Current alpha features:
- friend removal detection
- server disappearance detection
- group-DM disappearance detection
- startup/reconnect comparison
- delayed confirmation to reduce false alerts
- local history scaffolding
